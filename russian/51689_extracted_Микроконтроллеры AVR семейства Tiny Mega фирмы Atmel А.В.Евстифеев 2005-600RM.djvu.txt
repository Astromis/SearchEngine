Часть 1. Микроконтроллеры семейства Tiny 
Вообще говоря, «регистры» PINx регистрами не являются, по этим 
адресам осуществляется доступ к физическим значениям сигналов на 
выводах порта. Соответственно они доступны только для чтения, тогда 
как остальные регистры доступны и для чтения, и для записи. 
Таким образом, запись в порт означает запись требуемого состояния 
для каждого вывода порта в соответствующий регистр данных порта 
PORT*. А чтение состояния порта выполняется чтением либо регистра 
данных порта PORTjc, либо регистра выводов порта PINx. При чтении  
регистра выводов порта PIN* происходит считывание сигналов,  
присутствующих на выводах порта. А при чтении регистра данных порта PORT* 
происходит считывание данных, находящихся в регистре-защелке порта 
(как для входных, так и для выходных контактов). При нахождении  
микроконтроллера в состоянии сброса (см. Главу 3) выводы всех портов  
находятся в третьем состоянии (состояние высокого импеданса Hi-Z). 
4.3. Конфигурирование портов ввода/вывода 
Порты ввода/вывода (и даже отдельные разряды одноименных 
портов) разных моделей семейства имеют различные возможности 
по конфигурированию. Рассмотрение этих возможностей начнем с  
младших моделей семейства Tiny (ATtinyl lx, ATtinyl2x и ATtinyl5L), имеющих 
в своем составе единственный 6-разрядный порт ввода/вывода (порт В). 
Формат всех регистров, относящихся к этому порту, приведен 
на Рис. 1.46.
